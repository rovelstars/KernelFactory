# KernelFactory

🌽 Kernel Factory for RovelOS/Linux

This project automates the downloading and building of Linux kernels with specific patches and out-of-tree modules like NVIDIA drivers, tailored for RovelOS/Linux.

## Features

- Download and build specific Linux kernel versions
- Apply custom patches to the kernel
- Build and install out-of-tree modules like NVIDIA drivers

## Requirements

- Go 1.20+
- Make
- GCC
- Other build dependencies (e.g., libncurses-dev, bison, flex)
